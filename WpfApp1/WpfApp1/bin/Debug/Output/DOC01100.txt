3. Implementation

3.1. Outline of Operation

The basic mode of operation of TEXT involves copying words from input text to output, starting
newlines/newpages as necessary to avoid splitting words across lines and spacing the words across
a line to right justify them. If a warning character ('@') is encountered in the input text, the characters
which follow are decoded causing a modification to the basic mode of operation. Facilities controlled
in this way include indentation; margin and tabulation settings, underlining, centring, forced
newlines/peges and gaps for diagrams etc. Tables and diagrams can be built up by this means. In
addition library procedures are called directly when ''tiire'' is detected, allowing fiowcharts for example
to be generated within a document.
